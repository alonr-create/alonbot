התקדמות לפי שלבים

שלב 1: תשתית בסיסית הושלם

  • מבנה פרויקט TypeScript + ESM
  • Express server (port 3700) + health check
  • SQLite database (better-sqlite3, WAL mode)
  • Config loader (.env, dotenv)
  • Unified message interface (ChannelAdapter)
  • Gateway router (message dispatch)

שלב 2: חיבור ערוצים הושלם

  • Telegram adapter (grammy) - טקסט, תמונות, קול, מסמכים
  • Telegram commands (/start, /export, /menu)
  • Typing indicator בזמן עיבוד
  • WhatsApp adapter (Baileys) - מוכן, לא מחובר
  • WhatsApp QR pairing + חיבור לייב

שלב 3: מוח — Claude Agent הושלם

  • Claude API integration (claude-sonnet-4)
  • System prompt דינמי (עברית, עסקים, כלים)
  • Agentic tool loop (מקס 15 איטרציות)
  • Rate limiting (10 הודעות/דקה למשתמש)
  • Message truncation (4000 תווים)
  • Vision support (תמונות base64)

שלב 4: כלים (16 Tools) הושלם

  • shell - הרצת פקודות (24 פקודות מותרות)

  • web_search - חיפוש DuckDuckGo
  • browse_url - קריאת תוכן URL

  • send_voice - הודעה קולית (ElevenLabs v3)
  • remember - שמירת זיכרון (מובנה עם type/category/importance)
  • set_reminder / list_reminders / delete_reminder - תזכורות
  • monday_api - GraphQL queries ל-Monday.com
  • send_email - Gmail עם whitelist
  • manage_project - git status/log/pull/diff

שלב 5: Voice I/O הושלם

  • Auto-detect שפה (עברית/אנגלית)
  • Voice ID: Jessy Peres clone (עברית) + Brian (אנגלית)

שלב 6: Skills + Cron הושלם

  • Skill loader (Markdown files)
  • daily-brief.md - סיכום יומי 08:00
  • morning-greeting.md - ברכת בוקר 07:00
  • Cron scheduler (node-cron, Israel timezone)
  • Skills נוספים (דוחות דקל, חיפוש אינטרנט מתקדם) — עתידי

שלב 7: Deploy + Cloud/Local הושלם

  • Dockerfile (Node 22, multi-stage)
  • render.yaml - Render.com blueprint
  • Cloud mode (MODE=cloud) - Telegram polling + cron
  • Local mode (MODE=local) - cron + tool API
  • Cloudflare tunnel (cloud-to-local proxy)
  • UptimeRobot (anti-sleep ping)
  • Permanent named tunnel (URL changes on restart) — עתידי
  • LaunchAgent for auto-start on Mac — עתידי

שלב 8: אבטחה הושלם

  • Shell injection prevention (metachar blocking)
  • File path allowlist + .env/.git blocking
  • SSRF prevention (private IP blocking)
  • Email domain/address whitelist
  • Telegram user allowlist (empty = deny)
  • Rate limiting per user
  • System prompt anti-injection rules
  • API key in header (not URL)
  • Error messages sanitized

שלב 9: זיכרון מתקדם - פאזה 1 הושלם

  • טבלת memories (type, category, importance, source)
  • Smart retrieval (high-importance + keyword + category)
  • Category auto-detection (דקל, מצפן, אישי...)
  • Auto-summarize כל 40 הודעות
  • Conversation summaries injected to prompt
  • Migration from old facts table

שלב 10: זיכרון מתקדם - פאזה 2 (Vector Search) הושלם

  • sqlite-vec - vector search extension
  • Embedding pipeline (Gemini gemini-embedding-001, 768 dims)
  • Semantic search (cosine similarity < 1.2)
  • Memory vectors table (vec0 virtual table)
  • Auto-embed at startup (embedUnembeddedMemories)

שלב 11: זיכרון מתקדם - פאזה 3 (Maintenance) הושלם

  • Memory consolidation — מיזוג כפילויות
  • Importance decay (60+ ימים בלי גישה, importance < 8)
  • Duplicate merging (first 15 chars match)
  • Stale event cleanup (30+ ימים, importance ≤ 2)
  • Nightly cron 03:00 Israel time
