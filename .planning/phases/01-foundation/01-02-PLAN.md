       - Import makeWASocket, useMultiFileAuthState, DisconnectReason from @whiskeysockets/baileys
       - Import Boom from @hapi/boom
       - Import qrcode-terminal for terminal QR
       - Import QRCode from qrcode for web QR (toDataURL)
       - Module-level: sock (WASocket | null), retryCount = 0, MAX_RETRIES = 10

       - Export async function connectWhatsApp(): Promise<WASocket>
         a. Ensure session directory exists (config.sessionDir, mkdirSync recursive)
         b. const { state, saveCreds } = await useMultiFileAuthState(config.sessionDir)
         c. Create socket: makeWASocket({ auth: state, printQRInTerminal: true, browser: ['AlonDev Sales', 'Chrome', '22.0'] })
         d. sock.ev.on('creds.update', saveCreds)
         e. sock.ev.on('connection.update', handler):
            - If qr: generate data URL via QRCode.toDataURL(qr), call setQR(), set status 'connecting'
            - If connection === 'open': clearQR(), set status 'connected', reset retryCount to 0, log success, notify Alon "WhatsApp connected" (only on reconnect, not first connect)
            - If connection === 'close':
              * Set status 'disconnected'
              * Extract reason from (lastDisconnect?.error as Boom)?.output?.statusCode
              * If reason === DisconnectReason.loggedOut:
                - Delete session directory (rmSync recursive)
                - Notify Alon via Telegram: "WhatsApp session logged out -- QR re-scan needed at /qr"
                - Notify Alon via WhatsApp backup
                - Reconnect (will show new QR)
              * Else if retryCount < MAX_RETRIES:
                - Exponential backoff: delay = min(5000 * 2^retryCount, 60000)
                - Log retry attempt
                - setTimeout(connectWhatsApp, delay)
                - retryCount++
              * Else:
                - Notify Alon: "WhatsApp reconnection failed after 10 attempts -- manual intervention needed"
                - Log critical error
         f. Set up message handler (call setupMessageHandler)
         g. Store sock module-level, return sock

       - Export function getSocket(): WASocket | null
       - Export function getConnectionStatus() from qr.ts re-export
