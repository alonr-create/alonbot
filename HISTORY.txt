AlonBot — היסטוריית פיתוח מלאה
================================

AlonBot הוא עוזר אישי ועסקי מבוסס AI, בנוי ב-TypeScript + Node.js עם Claude API, SQLite, ו-Telegram.
פותח על ידי אלון עם Claude Code. נפרס על Render (ענן) עם auto-deploy מ-GitHub.
GitHub: alonr-create/alonbot

מצב נוכחי (מרץ 2026)
======================

Stack: TypeScript + Node.js ESM + SQLite + sqlite-vec + Claude API + Gemini API
Deploy: Render (Docker, auto-deploy from GitHub main)
GitHub: alonr-create/alonbot

32 כלים פעילים:
- מידע: web_search, web_research, browse_url, analyze_image
- יצירה: generate_image, send_voice
- זיכרון: remember, search_knowledge, learn_url, learn_text
- מערכת: shell, read_file, write_file, screenshot, manage_project, send_file
- עסקים: monday_api, send_email
- משימות: add_task, list_tasks, complete_task, api_costs
- יומן: calendar_list, calendar_add
- תזמון: schedule_message, set_reminder, list_reminders, delete_reminder
- אוטומציה: create_workflow, list_workflows, delete_workflow, toggle_workflow

Claude API Features (הכל מומש):
- Streaming Responses
- Extended Thinking (Sonnet 4 / Opus 4)
- Prompt Caching (system + multi-turn)
- Parallel Tool Execution
- Token Counting + Auto-Trim
- Citations (knowledge base)
- Batch API (50% savings)
- Vision (images + PDFs)
- Model Switching (Sonnet/Opus)
- Gemini Fallback
- Agentic Loop (15 iterations max)

DB Tables: messages, memories, memory_vectors, conversation_summaries,
  cron_jobs, api_usage, tasks, scheduled_messages, tool_usage,
  knowledge_docs, knowledge_chunks, knowledge_vectors,
  workflows, batch_jobs
