## 💬 איך הבוט מדבר עם לקוחות

### סטטוסים של ליד:
| סטטוס | משמעות |
|--------|--------|
| `חדש` | נכנס עכשיו, עדיין לא דיברנו |
| `נוצר קשר` | הבוט שלח הודעה ראשונה |
| `בשיחה` | שיחה פעילה עם הלקוח |
| `הצעה נשלחה` | הבוט נתן מחיר (₪) |
| `פגישה נקבעה` | נקבעה פגישת Zoom |
| `הועבר לטיפול` | הלקוח ביקש לדבר עם אלון |
| `נסגר ✅` | עסקה נסגרה! |
| `נסגר ❌` | לא הצליח |

### מתי הבוט מעביר לאלון?
- הלקוח כותב "תעביר לאלון", "רוצה לדבר עם אדם", "נציג"
- 3 הודעות קצרות ברצף (סימן לתסכול)
- הבוט מרגיש שהשיחה לא מתקדמת

### פולואפים אוטומטיים:
- **אחרי 24 שעות** → הודעת מעקב ידידותית
- **אחרי יומיים** → הודעה עם דחיפות
- **אחרי 4 ימים** → הודעה אחרונה
- פולואפ נשלח רק בשעות פעילות (א'-ה' 09:00-16:00, ו' 09:00-13:00)
- אם הלקוח עונה — הטיימר מתאפס
